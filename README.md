## ✨ Key Features

* **Dynamic Product Catalog:** Seamless browsing with advanced filtering and search.
* **Complete Shopping Flow:** Fully integrated cart management and checkout mechanics.
* **Secure Authentication:** User signup and login integration for a personalized experience.
* **Modern UI/UX:** A minimalist, prestigious design focus featuring smooth transitions and structured layouts.

---

## 🛠️ Tech Stack

* **Frontend:** Angular (HTML5, SCSS, TypeScript)
* **Hosting & Deployment:** Vercel
* **API Integration:** RESTful API for user authentication, products, and cart management
